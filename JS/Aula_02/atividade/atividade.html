Atividade da Aula 02

Receber os valores: nome (string), altura (number) e peso (number) em variáveis, calcular o IMC (peso / altura * altura) e mostrar no Console do navegador através do Javascript uma frase concatenada em Template String. Ex: “Meu nome é Fulano(a) possuo altura X e peso Y, meu IMC atual é de Z.” Subir arquivo no GitHub e enviar o link através do Moodle.
